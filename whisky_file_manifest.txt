
```bash

touch src/__init__.py src/data_loader.py src/validation.py src/preprocess.py

```

### 3. Install & Run
```bash

source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```
